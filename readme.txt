Add Viacompras payment gateway to WooCommerce.

== Description ==
**IMPORTANT: *WC - APG Viacompras payment gateway* required WooCommerce 2.1.0 or higher.**

**WC - APG Viacompras payment gateway** add to your WooCommerce the Viacompras payment gateway from BCP (Banco de Crédito del Perú).

== Installation ==
1. You can:
 * Upload the `wc-apg-viacompras-payment-gateway` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **WC - APG Viacompras payment gateway** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *WooCommerce -> Settings -> Checkout* or through *Settings* button.
4. Ready, now you can enjoy it, and if you like it and find it useful, make a [*donation*](https://artprojectgroup.es/tienda/donacion).
